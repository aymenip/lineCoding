About This Project

This project is intends to simulate the different Line Encoding Schemes used in real world. A line code is the code used for data transmission of a digital signal over a transmission line. This process of coding is chosen so as to avoid overlap and distortion of signal such as inter-symbol interference. Read More about types of Line Coding Schemes used in this project :

  • Unipolar
  • NRZ-L
  • NRZ-I
  • Polar RZ
  • Manchester
  • Differential Manchester
  • AMI
